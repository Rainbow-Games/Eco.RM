# Code Style
## Namespaces
### Eco.RM
Eco.RM is the namespace for all raynbo mods added files.
### Eco.RM.Components
Eco.RM.Components holds any World Object Component related data including their interfaces.
### Eco.RM.Items
Eco.RM.Items holds any items and item types and recipies.
Does not include WorldObjectItems.
### Eco.RM.Objects
Eco.RM.Objects holds any world objects their items and recipes.
### Eco.RM.Internal
Eco.RM.Internal holds any functions classes and what not that the user or other developers wont need to utilize it.
